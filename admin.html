Deploy to GitHub Pages
Step-by-step guide to host your portfolio for free.
1

Create a GitHub Repository

Go to github.com and create a new repository named username.github.io (replace "username" with your actual GitHub username). Make it public.

2

Download Your Portfolio Files

Download index.html, admin.html, and data.json from this admin panel using the save button above.

3

Upload Files to GitHub

Go to your repository → click "uploading an existing file" → drag and drop all three files → commit changes.

Or via Git CLI: git init git add . git commit -m "Initial portfolio" git branch -M main git remote add origin https://github.com/USERNAME/USERNAME.github.io.git git push -u origin main
4

Enable GitHub Pages

Go to Repository → Settings → Pages → Source: Deploy from branch → Branch: main → / (root) → Save.

5

Your Site is Live! 🎉

Visit https://username.github.io — it usually takes 1-2 minutes to go live.

📝 To update content: edit data.json on GitHub directly, or run admin.html locally, edit and download the JSON, then re-upload.
